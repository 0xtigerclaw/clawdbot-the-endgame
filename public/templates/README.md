# LinkedIn templates

Place your base LinkedIn image template(s) here.

## Default template

- Expected path: `mission-control/public/templates/linkedin_base.png`
- Used when the **Ive** step runs for a LinkedIn workflow: the system will overlay the core message text onto this image and write the result to `mission-control/public/generated/`.

## Custom path (optional)

Set `LINKEDIN_TEMPLATE_PATH` (absolute path recommended) to point to a different template file.
